## Description
DCA (Dollar Cost Averaging) planning tool allowing users to create multiple DCA plans for any coin, each linked to its own portfolio. Plans are generated by an LLM that produces 10 buy zones and 10 sell zones with probability scores (0–100). Users can execute, skip, or edit individual zones. Budget tracks deployed amount (sum of buy transactions only); sell proceeds are profit, not reclaimed budget.

## Main Flow
1. User visits `/dca` — sees list of all DCA configs as cards, each showing coin, capital summary (deployed/remaining), and pending zone counts.
2. User clicks "New Plan" — dialog opens requesting Portfolio Name (auto-creates new Portfolio), Coin (BTC/ETH), and Budget (USD).
3. Submit creates a new Portfolio and DcaConfig via `POST /dca/config`, redirects to `/dca/[configId]`.
4. On detail page, user clicks "Generate Plan" — LLM fetches candles, analyzes capital state, produces 10 buy zones + 10 sell zones with probability scores.
5. Zones displayed in two sections: BUY ZONES and SELL ZONES. Each zone shows target price, amount, probability badge (0–39: red, 40–69: amber, 70–100: green), status, and actions.
6. User can execute (mark as done + create portfolio transaction), skip, edit, or delete zones.
7. User can "Re-analyze" (LLM updates analysis text only) or "Re-plan" (archives current plan, generates new one).
8. Note rows are expandable in the table (click row to expand/collapse).

## Edge Cases
- If both `portfolioName` and `portfolioId` are provided in new plan request → 400 error (must use one or the other).
- If plan generation fails → error shown inline, existing plan unchanged.
- Multiple DCA configs per coin allowed (each with its own portfolio).
- Clicking a card with no active plan still navigates to detail page; user can generate from there.
- Notes are hidden by default; click row to expand/collapse.

## Budget Logic
- `deployedAmount = sum(buy transactions only)`
- `remaining = totalBudget - deployedAmount`
- Sell proceeds are tracked as profit in the portfolio, not added back to DCA budget.
